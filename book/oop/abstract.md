## Абстрактные классы, чисто виртуальные функции 


к чисто виртуальным функция относится те функции которые нужны чтобы их переопределил


```
class AbstractParent{
    
    // = 0 значит что нужно переделать иначе будет ошибка компиляции
    virtual void func() = 0; 
    virtual void get() const = 0;
};

```
