## Деструктор 

*Деструктор* - функция которая вызывается при уничтожении обьекта
нужен для очистки памяти.Пишется с `~` знака

## Вызывается при:
* Уничтожения контейнера в котором находится переменная
* Выходе из зоны видимости стека
* Ручного удаления delete 



## Синтаксис
```

// Human - название класс
~Human(){
    // код для очистки
}

```
