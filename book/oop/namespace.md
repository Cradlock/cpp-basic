## Namespace
`namespace` - это команда для создания `пространства имен`.
Это способ изолировать и разделить код на кусочки.


Допустим нам нужны функции для определенной логики:

```
void sum(...){

}
```
Но где то в коде есть еще такая же функция но для чего то еше

```
void sum(...){

}
```

И вот теперь у нас две функции и комплиятор не знает какой использовать.
Поэтому нужны `namespace`

```
namespace Finance{
    void sum(){}
}


namespace Mathing{
    void sum(){}
}

```
Теперь к ним можно получить доступ так

```
Finance::sum();

Mathing::sum();
```

Также есть возможность создать анонимное пространство имен доступное только в одном 
файле.


```
namespace {
    ...
}
```

Если сделать так то переменные и функции внутри namespace не смогут увидет другие файлы
