## Статические аттрибуты в класса c++


**static** - Модификатор, делает аттрибут класса переменную  или функцию делает общей для всех 

**inline** - Компилятор буквально копирует функцию туда где ее вызывают 

```
class Human {
    public:


        Human (string new_name) : name(new_name) {
            count++;
        }
    
    // Статическая функция 
    static int getCount() {
        // Обращение к статическим аттрибутам (можно просто через обьект)
        return Human::count;
    }

    private:
        // `inline` - нужен чтобы при использовании функции они сразу получали значение 
        
        // `static` - нужен чтобы переменная `count` был общим для всех обьектов и принадлежал классу 
        static inline int count = 0;
        string name;

```


## Использование 

```
Human bob("BOB");

// Можно и так
Human::getCount();

// Можно и так 
bob.getCount();


```
