## array

`array` - контейнер с фиксированным размером.Но зато быстрее и экономичнее чем `vector`


### Заголовок

```
#include <array>
```


### Инициализация
```
std::array<int,5> numbers; // массив с 5 элементов типа int 
```


### Инициализация
```
std::array<int,10> numbers = {1,2,42,4,1}; // Больше элементов = ошибка
```

### Получение элементов
```
numbers[index]; // Может быть ошибка если за пределы
```
--- 

* `at`
```
numbers.at(index); // Бросает исключение если не найдено
```

---

* `front/back`
```
numbers.front() // Первый элемент
numbers.back(); // Последний элемент
```

### Изменение

* `fill`
```
numbers.fill(value); // меняет все значения на value 
```
