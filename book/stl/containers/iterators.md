## Итераторы

`итератор` - в рамках `STL`, это умный указатель с которым работает пакет
`algorithm` а он включает в себя поиск,сортировку,фильтрацию и т.д


## Получение 
Итератор можно получить из любого контейнера
```
vector<int> v = {1,2,2,3}; // вот контейнер

auto iter = v.begin(); // указывает на начало

auto iter = v.end(); // указывает на конец 

auto iter = v.rbegin(); // Отдает реверсивный итератор это конец

auto iter = v.rend(); // Реверсивный итератор начало

auto iter = v.cbegin(); // Константный итератор нельзя менять 

auto iter = v.cend(); // Конастантный итератор нельзя менять
```


## Использование 

С помошью итераторов можно делать несколько действий:

- `*iter`: Получения элемента на который указывает итератор

- `++iter`: Сдвигает указатель на следующие элемент в массиве

- `--iter`: Сдвигает на предыдущий элемент

- `iter1==iter2 / iter1 != iter2`: Проверка двух указателей, указывают ли они на один и тот же элемент

- `iter {+n/-n}`: Возвращает сдвинутый итератор

- `iter {+=n/-=n}`: Перемещает итератор на `n` позицию

- `>, >=, <, <=`: операции сравнения: 
Один итератор больше другого, если указывает на элемент, который ближе к концу


Не все контейнеры поддерживают все операции.Так как у них разная структура.
