## vector

`vector` - один из самых используемых контейнеров, является динамическим.Можно хранить данные одного типа
который указывается при создании вектора.
```
vector<int> ...;
```


## Инструкция

### Заголовок
```
#include <vector>
```


### Инициализация

Есть несколько способов инициализации:



1. Пустой вектор 
```
std::vector<int> v1;
```

2.  Копия другого вектора
```
std::vector<int> v2(v1);

std::vector<int> v2 = v1;
```

3. Вектор в котором заранее указан размер и значения
```
std::vector<int> v1(5); // ->  теперь он хранит 5 чисел каждое равно 0

std::vector<int> v2(5,2); // -> хранит 5 чисел каждое равно 2

std::vector<int> v3{1,2,3,4,5,6,7}; // Уже хранятся данные: 1,2,3,4,5,6,7

std::vector<int> v4 = {1,2,3,4,5,6}; // Тоже данные уже хранятся
```
    

## Получение элементов

1. По индексу
```
v[index]; // По индексу

v.at(index); // Тоже по индексу, только вот если нет элемента бросает исключение `std::out_of_range`

v.front(); // Первый элемент

v.back(); // Последний элемент

```

## Изменение 

### Добавление в конец:

* `push_back` - добавляет готовый обьект при  этом копируя его

```
v.push_back(12); // Добавить элемент в конец
```

* `emplace_back` - создает обьект на лету и ставит в конец

```
v.emplace_back(12); // Добавит элемент в конец

// можно также указать параметры если создаеш обьект
v.emplace_back(Args...); // аргументы для твоего класса
```

### Добавление в определенную позицию:
Для последующих элементов нужен `итератор`

* `emplace` 
```
v.emplace(it+2,23); // ставит после второго элемента
// Также можно указать параметры если добавляеш класс
v.emplace(it+2,Args...);
```
* `insert`
```
v.insert(it+2,56); // ставит элемент после второго,ставит готовый элемент 
```
---
```
v.insert(it+2, 4, 56); // ставит 4 `56` после второго элемента
```
---
```
v.insert(it+3, other_b_it,other_e_it); // ставит в определенное место данные из другого массива
```
---  
```
v.insert(it+3,{values...}); // ставит в `it+3` позицию values элементы
```

### Удаление:
* `clear`
```
v.clear(); // очистить
```
* `pop_back`
```
v.pop_back(); // Удаление последнего
```
* `erase`
```
v.erase(it); //  удаляет то на что указывает итератор it 
```
--- 
``` 
v.erase(begin,end); // удаляет целый диапазон
```

### Изменение 
* `assign`
```
v.assign(4, "New"); // заменяет все элементы на опредленное количество элементов
```
---
```
v.assign({"New","Old","Your"}); // заменяет все элементы на новый массив
```
* `swap`
```
v.swap(other_v); // поменять значения двух векторов
```
